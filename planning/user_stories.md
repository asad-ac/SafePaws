# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

[👉🏾👉🏾👉🏾 Include at least at least 1, but no more than 3, user roles.]

As a sanctuary staff member, I want to manage animal records, so that I can efficiently support daily animal care, sponsor relationships, and volunteer coordination.

## Draft User Stories

[👉🏾👉🏾👉🏾 Include at least at least 10 user stories in this format:]

1. As an Admin, I want to add an animal, so that I can manage its availability, personal information, and health status.
2. As an Admin, I want to update an animal's health status, name, weight, vaccination status, and tags, so that I can ensure accurate care records.
3. As an Admin, I want to remove an animal that has been transferred to another sanctuary or has passed away from our system, so that I can maintain an up-to-date list of animals.
4. As an Admin, I want to update tags (friendly, vaccinated, healthy) so that, I can better track each animal’s condition and needs.
5. As an Admin, I want to delete tags that are no longer applicable, to maintain an updated and organsized system.
6. As an Admin, I want to view all the animals in a grid so that, I can scan through and organize next steps. 
7. As an Admin, I want to filter animals by care status, feeding status, and enclosure cleaning status to know what to follow up on.
8. As an Admin, I want to sort animals by intake date, name, and age.
9. As an Admin, I want to generate a post to ask for donations or volunteers so that I can ensure care.
10. As an Admin, I want to add sponsors, so that I can track and manage sponsor relationships.
11. As an Admin, I want to update sponsor information to ensure accurate information.
12. As an Admin, I want to remove sponsors to keep sponsorship records up to date and identify animals available for sponsorship.
13. As an Admin, I want to add a volunteer to assign them to daily care tasks. 
14. As an Admin, I want to update a volunteer to adjust their assigned daily care volunteer. 
15. As an Admin, I want to remove a volunteer so that I can keep records up to date when a volunteer leaves or an animal is no longer in the system.
